## Danh sách tệp nguồn
Kho lưu trữ chứa mã nguồn cơ bản của dự án để đảm bảo tính tương thích trên các hệ thống khác nhau:
- CPU_Scheduler.pro: Tệp quản lý cấu hình dự án của Qt.
- main.cpp: Tệp khởi tạo chương trình.
- mainwindow.cpp / mainwindow.h: Logic xử lý giao diện và thuật toán.
- mainwindow.ui: Tệp thiết kế giao diện người dùng.

## Yêu cầu hệ thống
Để biên dịch và vận hành dự án, máy tính cần cài đặt:
- Qt Creator (Phiên bản 5.15 hoặc 6.x trở lên).
- Trình biên dịch MinGW hoặc MSVC đã được cấu hình tích hợp.

## Hướng dẫn khởi chạy dự án
1. Tải toàn bộ thư mục về máy tính hoặc sử dụng lệnh Git Clone.
2. Mở phần mềm Qt Creator.
3. Chọn "Open File or Project" và tìm đến tệp CPU_Scheduler.pro.
4. Tại giao diện "Configure Project", chọn bộ công cụ biên dịch (Kit) tương ứng trên máy tính và nhấn "Configure Project".
5. Nhấn tổ hợp phím Ctrl + R để bắt đầu quá trình biên dịch tự động và khởi chạy ứng dụng.
